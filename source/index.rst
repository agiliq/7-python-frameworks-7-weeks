Seven Python Web frameworks in Seven Weeks
============================================

This book is about seven python webframeworks which are being extensively used. Throughout this book we will be describing each framework with the :code:`use cases`, :code:`installation`, and a :code:`demoapp` for each framework. We will also include few applications built using the framework followed by some pros and cons. There is lot to learn in this book. This book is for people with intermediate knowlwdge of Python.
